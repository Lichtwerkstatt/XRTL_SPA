# SliderCtrl(component, title, min, max, sliderValue, online, option) → {React.ReactElement}

Used to create a slider with a specified start and end point. In addition, the controlId, the status (of the component and the online status) and the command for the server must be specified. The title can but does not have to be given.

# StepperCtrl(component, length, left, right, buttonValue) → {React.ReactElement}

This component returns a styled stepper. For this, the controlId, the length of the stepper and the beginning of the Id of the paragraphs must be specified. Optionally, the text on the right and left can be defined.

Parameters:
Name Type Description
component string

controlId
length number

Specifies the length of the stepper
left string

text at the left/beginning
right string

text at the right/end
buttonValue number

indicates the beginning of the Ids of the paragraphs of the text (these should be set once at 1, 10, 20, ..., because otherwise, when opening several windows, all elements with the same ID will be changed).)

# SwitchCtrl(component, left, right, switchStatus, online, option) → {React.ReactElement}

This component returns a styled switch. The left/beginning and right/end text of the switch can be set. In addition, the controlId, the status (of the component and the online status), and the command for the server must be specified.
