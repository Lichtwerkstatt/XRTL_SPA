# SliderCtrl(component, title, min, max, text, sliderValue, online, option) → {React.ReactElement}

Used to create a slider with a specified start and end point. In addition, the controlId, the status (of the component and the online status) and the command for the server must be specified. The title can but does not have to be given. To display text under the slider instead of numbers, this can be passed to the slider within the tag text in the form of an array.

Parameters:
Name Type Description
component string

controlId
title string

title
min number

text at the left/beginning
max number

text at the right/end
text array

contains an array with numbers as keys and strings as values (for displaying text under the slider instead of numbers)
sliderValue number

Status of the slider from the status request
online boolean

connection status to the component
option string

command for the server

# StepperCtrl(component, length, left, right, buttonValue) → {React.ReactElement}

This component returns a styled stepper. For this, the controlId, the length of the stepper and the beginning of the Id of the paragraphs must be specified. Optionally, the text on the right and left can be defined.

Parameters:
Name Type Description

controlId
length number

text at the left/beginning
right string

text at the right/end
buttonValue number

indicates the beginning of the Ids of the paragraphs of the text (these should be set once at 1, 10, 20, ..., because otherwise, when opening several windows, all elements with the same ID will be changed).)
